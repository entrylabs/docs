GitHub fork 방법

GitHub의 소스를 이용할 때에는 GitHub의 fork 기능을 사용하여 저장소를 복사해서 자신의 저장소에서 작업한 후 원래 저장소에 pull request를 생성하는 것을 권장합니다. 하지만 해당 기능을 잘 모르는 경우에는 clone 기능을 이용하여 저장소를 복사하여 작업한 후 작업한 소스를 엔트리에 전달해도 됩니다.
