자료 블록

1. (안녕!) 을(를) 묻고 대답 기다리기

이 블록에 입력한 내용()을 말풍선으로 묻고, 실행 화면에 아래와 같이 대답을 입력할 수 있는 창이 나타납니다.

대답을 입력할 수 있는 이 창을 ‘대답창’이라고 부를게요.

2. 대답

대답 창에 입력한 내용을 가져오는 값 블록입니다.

3. 대답 [숨기기]

대답을 실행 화면에서 숨기거나 나타냅니다.

목록 상자()를 클릭하면 대답을 숨길지 나타낼지 고를 수 있어요.

4. [변수] 값

변수의 내용을 가져오는 값 블록입니다.

목록 상자()를 클릭하면 내용을 가져올 변수를 선택할 수 있어요.

5. [변수] 에 (10) 만큼 더하기

아래와 같이 동작하는 블록입니다.

  • 변수의 내용과 입력한 내용()이 숫자일 때 : 변수의 숫자에 입력한 숫자를 더합니다.
  • 변수의 내용과 앱력한 내용() 중 하나라도 문자일 때 : 변수의 내용에 입력한 내용을 뒤에 추가합니다.

목록 상자()를 클릭하면 더하거나 추가할 변수를 선택할 수 있어요.

6. [변수] 를 (10) (으)로 정하기

변수의 내용을 입력한 내용()으로 정합니다.

목록 상자()를 클릭하면 내용을 정할 변수를 선택할 수 있어요.

7. 변수 [변수] 보이기

변수를 실행 화면에서 보이게 합니다.

목록 상자()를 클릭하면 보일 변수를 선택할 수 있어요.

8. 변수 [변수] 숨기기

변수를 실행 화면에서 숨깁니다.

목록 상자()를 클릭하면 숨길 변수를 선택할 수 있어요.

9. [리스트] 의 (1) 번째 항목

리스트에서 입력한 순서()의 항목을 가져오는 값 블록입니다.

목록 상자()를 클릭하면 항목을 가져올 리스트를 선택할 수 있어요.

10. (10) 항목을 [리스트] 에 추가하기

입력한 내용()을 리스트의 마지막 항목으로 추가합니다.

목록 상자()를 클릭하면 항목을 추가할 리스트를 선택할 수 있어요.

11. (1) 번째 항목을 [리스트] 에서 삭제하기

입력한 순서()의 항목을 리스트에서 삭제합니다.

목록 상자()를 클릭하면 항목을 삭제할 리스트를 선택할 수 있어요.

12. (10) 을(를) [리스트] 의 (1) 번째에 넣기

입력한 내용()을 리스트에 입력한 순서()의 항목으로 추가합니다.

목록 상자()를 클릭하면 항목을 추가할 리스트를 선택할 수 있어요.

13. [리스트] (1) 번째 항목을 (10) (으)로 바꾸기

리스트에서 입력한 순서()의 항목을 입력한 내용()으로 바꿉니다.

목록 상자()를 클릭하면 항목의 내용을 정할 리스트를 선택할 수 있어요.

14. [리스트] 항목 수

리스트가 가진 항목 개수를 가져오는 값 블록입니다.

목록 상자()를 클릭하면 항목 개수를 가져올 리스트를 선택할 수 있어요.

15. [리스트] 에 (10) 이 포함되어 있는가?

리스트의 항목 중에서 입력한 내용()과 같은 항목이 있다면 참으로, 아니라면 거짓으로 판단하는 블록입니다.

목록 상자()를 클릭하면 항목을 비교할 리스트를 선택할 수 있어요.

16. 리스트 [리스트] 보이기

리스트를 실행 화면에서 보입니다.

목록 상자()를 클릭하면 나타낼 리스트를 선택할 수 있어요.

17. 리스트 [리스트] 숨기기

리스트를 실행 화면에서 숨깁니다.

목록 상자()를 클릭하면 숨길 리스트를 선택할 수 있어요.
